Large-Scale Evolutionary Algorithms


Evolutionary Algorithms (EA) are a family of optimization algorithms based on the thoery of Natural Selection. Optimizing high-dimensional problems is a difficulty for all algorihms and EA's are no acception, this page provides works addressing this issue in the EA field.

    Decomposition Based Methods


    Decomposition based methods handle high-dimensional problems by splitting the original problem in several lower-dimensional problems. Each lower-dimensional problem is then optimizes using an Evolutionary Algorithm in turn. The core focus of these decomposition methods is the method of forming the sub-problems, evaluating each sub-problem and how much resource should be given to different sub-problems (some sub-problems may have larger impact on the original problem than others)
